# Contributing

Thanks for helping improve ArchSql.

## Development setup

- Install the .NET 10 SDK.
- Build the tool:
  - `dotnet build src/ArchSql/ArchSql.csproj`
- Run the test suite:
  - `dotnet test tests/ArchSql.Tests/ArchSql.Tests.csproj`

## Pull request expectations

- Keep changes focused and well described.
- Add or update tests when behavior changes.
- Avoid committing secrets, credentials, or local-only artifacts.

## Reporting issues

Please open a GitHub issue with steps to reproduce, expected behavior, and relevant logs.
